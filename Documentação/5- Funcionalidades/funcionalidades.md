# Funcionalidades do Sistema 
As funcionalidades representam os serviços e operações que o sistema oferece a seus usuários. Elas foram definidas com base nas necessidades identificadas nas entrevistas, no contexto do problema e no fluxo real da coleta de lixo verde no município. A seguir, descrevem-se detalhadamente as funcionalidades principais:


| Código | Funcionalidade | Descrição |
|--------|----------------|-----------|
| F1 | Cadastro de Itens de Lixo Verde | Permite ao cidadão registrar resíduos a serem coletados, informando descrição, categoria (folhas, galhos, poda), quantidade aproximada e localização. |
| F2 | Remoção de Itens Cadastrados | Possibilita ao gestor ou coletor excluir solicitações já atendidas ou registros duplicados, mantendo apenas dados atualizados no sistema. |
| F3 | Agendamento de Coleta | Permite que o usuário agende a coleta conforme o cronograma municipal, contribuindo para melhor organização logística das equipes. |
| F4 | Envio de Imagens | Permite anexar fotografias dos resíduos, auxiliando a equipe de coleta na identificação visual do volume, tipo e localização exata. |
| F5 | Denúncias de Descarte Irregular | O cidadão pode relatar ocorrências de descarte inadequado em vias públicas, anexando fotos e localização para análise do gestor. |
| F6 | Visualização do Mapa por Zona | Exibe setores da cidade, mostrando solicitações pendentes, denúncias e pontos já atendidos para melhor tomada de decisão. |
| F7 | Login e Gerenciamento de Usuários | Controla o acesso ao sistema com níveis diferenciados para cidadãos, coletores e gestores, garantindo permissões específicas. |
| F8 | Módulo de Comunicação | Envia notificações e mensagens automáticas sobre status das solicitações e previsões de coleta. |
| F9 | Registro de Coleta Realizada | O coletor registra diretamente no sistema quando a coleta é concluída, atualizando o status das solicitações em tempo real. |
